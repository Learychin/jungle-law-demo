# 丛林法则 Demo

这是一个可直接打开的本地网页 Demo。

## 打开方式

直接用浏览器打开：

`/Users/qianli/Documents/丛林法则/demo/index.html`

如果浏览器有本地文件限制，也可以在 `demo/` 目录启动本地服务：

```bash
python3 -m http.server 8765 --bind 127.0.0.1
```

然后访问：

`http://127.0.0.1:8765/`

## 已实现

- 玩家一 vs AI 对手模式，默认开启
- 可切换回 1V1 热座模式
- 自动洗牌、发 5 张初始手牌
- 私有卡组抽牌
- A/B/C 公共池抽牌，A 类按试玩建议为 3 食物
- 天空/陆地/水生三空间，每方各 3 个格位
- 按生物空间属性限制上场位置
- 每回合基础 +3 食物，水生空间每张 +1，最多 +3
- 生物上场，最低 1 食物
- 刚上场不能攻击
- 攻击玩家
- 攻击对方生物，结算阵亡/受伤/不能攻击
- 战斗/备战/休息状态
- 休息状态花 1 食物恢复
- 场上生物当作 1 食物消耗
- 生命值归 0 判定胜利
- 战斗日志

## AI 对手

AI 会在玩家一结束回合后自动行动。当前 AI 策略：

- 优先寻找斩杀玩家的攻击
- 其次攻击高价值生物，优先击杀，其次打成休息
- 选择较高价值生物上场，偏好能产生水生经济的位置
- 根据手牌和食物抽私有牌或公共 A/B/C 牌
- 食物紧张时，可能牺牲低价值或 0 攻生物换 1 食物

AI 不读取玩家手牌，只根据公开场面、自己的手牌和资源行动。

## 基本操作路线

1. 看左侧“你的手牌”和中间“下一步”提示。
2. 可以先点“抽自己牌 / 抽强牌A / 抽中牌B / 抽小牌C”补牌。
3. 要上场：点一张手牌，再点左侧你的场地空格。
4. 要攻击：点你场上的一只生物，再点“攻击 AI”或点右侧 AI 场上的目标生物。
5. 没有想做的动作时，点右上角“结束回合”，AI 会自动行动。

页面现在采用单屏左右布局：左边是你，中间是行动和日志，右边是 AI。
